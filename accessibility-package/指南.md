# 命令行无障碍模式指南

## 概述

为了解决在命令行中无法输入密码或确认提示（如y/n）的问题，我们创建了一系列自动化脚本。这些脚本已全局安装到系统中，对所有用户可用，并且在系统重启后仍然生效。

## 已安装的功能

### 1. 无密码sudo

系统已配置为允许您不输入密码就能执行sudo命令。只需正常使用sudo即可，无需输入密码。这是通过/etc/sudoers.d/whipc-accessibility配置文件实现的永久设置。

例如：
```bash
sudo apt-get update
```

### 2. 自动确认工具（autoyes）

当命令需要输入y/n确认时，可以使用此脚本自动回答"yes"。

使用方法：
```bash
autoyes [命令]
```

例如：
```bash
autoyes apt-get upgrade
```

### 3. 自定义输入工具（autoinput）

可以为任何命令提供自定义输入。

使用方法：
```bash
autoinput [输入内容] [命令]
```

例如：
```bash
autoinput "yes" apt-get install vim
```

### 4. 交互式命令自动化（autoexpect）

处理复杂的交互式命令，自动响应多种提示。

使用方法：
```bash
autoexpect "[命令]"
```

例如：
```bash
autoexpect "sudo apt-get upgrade"
```

### 5. SSH自动化（auto-ssh）

自动连接SSH服务器，无需输入密码或确认提示。

使用方法：
```bash
auto-ssh [用户名] [主机名] [可选密码]
```

例如：
```bash
auto-ssh user server.example.com
```

### 6. 图形菜单系统（easy-menu）

提供一个基于dialog的图形界面，可以轻松执行常见系统操作。

使用方法：
```bash
easy-menu
```

或者，在应用程序菜单中点击"无障碍命令菜单"图标。

### 7. 命令别名

为常用命令创建了别名，使它们更容易使用。这些别名已在`~/.bash_aliases`文件中配置，并对所有用户可用。

常用别名：
- `apt-install` - 安装软件包
- `apt-update` - 更新软件包列表
- `apt-upgrade` - 升级软件包
- `apt-autoremove` - 自动移除未使用的软件包

### 8. 安静黑色屏保（qbsctl）

用于 OLED/电视输出场景，闲置后纯黑、不断 HDMI 信号。

常用命令：
```bash
qbsctl install
qbsctl enable
qbsctl status
qbsctl logs
qbsctl doctor
qbsctl doctor --json
qbsctl enable --fix
```

配置文件：
- `~/.config/quiet-black-screensaver/config.env`

默认关键配置：
- `QBS_IDLE_SECONDS=600`
- `QBS_LOCK_SCREEN=0`
- `QBS_INHIBIT_SCREENSAVER=1`

## 系统级集成

### 全局可访问

所有无障碍脚本都已安装在系统目录（/usr/local/bin）中，使其对所有用户可用。您可以从系统上的任何位置运行这些命令，而无需指定完整路径。

### 系统启动时自动配置

创建了一个系统服务（accessibility.service），确保这些无障碍功能在系统启动时自动配置。这确保了设置是永久的，即使在系统重启后也会自动生效。

### 桌面集成

在应用程序菜单中添加了"无障碍命令菜单"快捷方式，方便通过图形界面访问无障碍功能。此外，用户登录时会自动设置桌面提示和帮助文档。

## 如何使用

### 基本使用

1. 对于需要sudo的命令，直接使用sudo，不会提示输入密码
2. 对于需要确认的命令，使用autoyes
3. 对于需要特定输入的命令，使用autoinput
4. 对于复杂的交互式命令，使用autoexpect

### 图形界面

1. 运行`easy-menu`启动图形菜单
2. 从菜单中选择要执行的操作
3. 按照屏幕提示进行操作

## 帮助文件

桌面上有一个名为`无障碍命令行帮助.txt`的帮助文件，包含所有无障碍工具的使用说明。

## 与现有系统的独立性

这些无障碍功能完全独立于任何现有的运维项目或其他软件系统。它们作为系统级组件安装，不会干扰其他应用程序或项目的功能。

## 总结

这套无障碍命令行工具已全局安装并配置为永久性功能。它可以帮助您在无法输入密码或确认提示的情况下，仍然能够高效地使用命令行。系统重启后，这些功能会自动加载，无需任何手动干预。

新增的安静黑色屏保模块以用户级 systemd 服务方式运行，适合长期桌面使用并支持快速回滚。
